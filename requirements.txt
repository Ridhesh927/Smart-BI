# Project Dependencies

This file serves as a consolidated list of dependencies for the Smart Dash project.

## Backend Dependencies (Node.js/Express)
- cors: ^2.8.6
- dotenv: ^17.4.2
- express: ^5.2.1
- firebase-admin: ^13.8.0
- multer: ^2.1.1
- mysql2: ^3.22.0
- razorpay: ^2.9.6

## Frontend Dependencies (React/Vite)
- firebase: ^12.12.0
- framer-motion: ^12.38.0
- lucide-react: ^1.8.0
- react: ^19.2.4
- react-dom: ^19.2.4
- react-router-dom: ^7.14.1

## Installation Command
To install all dependencies, run the following commands in their respective directories:

### For Backend:
cd backend
npm install

### For Frontend:
cd frontend
npm install
